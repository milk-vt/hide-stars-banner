初心者が書いたフィルターになりますので問題などありましたらissue飛ばしてください

# そもそも広告枠を削除すれば良いのでは？
広告枠自体を削除する場合はマイフィルターに下記を追加するだけです。
```schedule.hololive.tv###carouselExampleIndicators```
これであればフィルターの追加は不要です。
